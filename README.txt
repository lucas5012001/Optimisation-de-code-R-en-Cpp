Dans ce code, j'ai programmé trois versions de l'algorithme des k-means :

- Une première version non optimisée, codée en R.

- Une seconde version, optimisée en C++ (Rcpp).

- Une troisième version, optimisée en C++ et parallélisée.

Ensuite, je compare l'efficacité de ces trois algorithmes.

Le code le plus rapide est la seconde version, qui est environ 250 fois plus efficace que les deux autres.
